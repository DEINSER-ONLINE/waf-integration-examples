# Deinser WAF Integration Examples

<p align="center">
  <img src="https://waf.deinser.com/images/logo/logo_256.png" width="256" alt="Deinser WAF Logo">
</p>

**Deinser WAF** is a Web Application Firewall (WAF) tool that enables you to protect websites by setting blocking rules based on IP, CIDR, country, ASN, and User-Agent. It includes a web dashboard for managing websites, global and site-specific rules, traffic statistics, blocked IPs, and an action log. The API allows you to integrate your application to send request logs and retrieve the list of blocked IPs in real time.

You can find more information at https://waf.deinser.com
